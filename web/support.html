What browsers and devices are supported? +
Gravitation¬≥ works on modern web browsers with WebGL support: Chrome, Firefox, Safari, and Edge. Mobile support includes iOS Safari and Android Chrome. For optimal performance with smooth animations and trails, a dedicated GPU and 4GB+ RAM is recommended. The application is responsive and adapts to different screen sizes automatically.

Can I use the AI models for training and research? +
Yes! All AI models are trained and ready to use. You can run the Python training scripts to gather additional simulation data and continue training the models, adding to their capabilities. This allows for extended research and personalized model development based on your specific simulation parameters.
How do I set up the servers for AI integration? +
Navigate to the Documentation and Guides page for complete setup instructions. You'll need to start the LLM Chatbot Server (Port 5001), Data Collection Server (Port 5002), and optionally the AI Model Server (Port 5003). Detailed step-by-step commands are provided in the documentation.
