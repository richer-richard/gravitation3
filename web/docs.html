üöÄ Quick Start (No Setup Required)

Gravitation¬≥ works entirely in your browser with no installation needed!

  1. Open any simulation HTML file in a modern browser
  2. Click "Start" to begin the simulation
  3. Adjust parameters in real-time
  4. Watch as AI makes predictions (if servers running)

Supported Browsers: Chrome, Firefox, Safari, Edge (modern versions)

üíª Developer Setup

For AI model training and advanced setup:

# Prerequisites
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r scripts/python_training/requirements.txt
npm install (for model management)

Requires: Python 3.11+, Node.js 16+

ü§ñ AI & Data Servers

To enable AI predictions and data collection:

# Terminal 1: LLM Chatbot Server (Port 5001)
cd api && ./start_llm_server.sh
# Output: http://localhost:5001/api/chat

# Terminal 2: Data Collection Server (Port 5002)
cd api && ./start_data_server.sh
# Output: http://localhost:5002

# Terminal 3: AI Model Server (Port 5003, optional)
cd api && ./start_model_server.sh
# Output: http://localhost:5003/api/health

Optional: Set OPENAI_API_KEY for GPT-5 chatbot

üìä Training Data Collection

Collect training data for AI models:

# 1. Start data server
./api/start_data_server.sh

# 2. Open simulation in browser
# e.g., lorenz-attractor/sim.html

# 3. In browser console (F12):
window.lorenzAI.startDataCollection()

# 4. Let simulation run 10+ seconds
window.lorenzAI.stopDataCollection()

# Data saved to: ai_data/lorenz-attractor/

üß† Training AI Models

Train neural network models with collected data:

# Train individual model
cd scripts/python_training
python train_lorenz.py

# Or train all models
python train_all_simulations.py

# Models saved to:
# ai_models/{simulation}/

  • 5001 - LLM Chatbot Server (GPT-5 chat)
